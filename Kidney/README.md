# Chronic-Kidney-Disease-Prediction
The goal of this project is predict the chronic kidney disease using parameters.

## Setup Instructions

1. Install dependencies:
   `pip install -r requirements.txt`

2. Train the model (Generates kidney_model.pkl using XGBoost):
   `python model.py`

3. Run the Web Application:
   `python app.py`

4. Open browser at http://127.0.0.1:5000
