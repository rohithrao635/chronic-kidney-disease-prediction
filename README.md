# Chronic Kidney Disease Prediction
## Overview
This project predicts whether a patient is likely to have Chronic Kidney Disease using Machine Learning.

## Features
- Data preprocessing
- Feature engineering
- Machine Learning model training
- Prediction interface
- User-friendly web application

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask/Streamlit (whichever you used)
- HTML
- CSS

## Dataset
UCI Chronic Kidney Disease Dataset

## Machine Learning Algorithm
(Random Forest / Decision Tree / Logistic Regression / XGBoost)

## Installation
pip install -r requirements.txt

## Run
python app.py
